# Sanguosha-WebArena 项目工作报告

> **项目概述**：基于 Spring Boot + React 的在线桌游「三国杀」Web 对战平台，支持用户注册登录、房间创建、WebSocket 实时对战的 1v1 卡牌游戏。

---

## 一、架构概览

| 层级 | 技术栈 | 说明 |
|------|--------|------|
| **后端** | Spring Boot 3.2 + Java 21 + Maven | RESTful API + WebSocket 实时通信 |
| **数据库** | MySQL + MyBatis-Plus | 用户数据持久化，游戏状态内存管理 |
| **前端** | React 18 + TypeScript + Vite | SPA 单页应用，状态管理用 Zustand |
| **通信** | JWT Token 认证 + WebSocket 长连接 | 登录鉴权后建立实时通信通道 |
